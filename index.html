Trip brief

最终环岛规划总览

需要和大家讨论的内容：航班衔接、一些细节路线、活动预订、行李和歌单都集中在这里。

航班

Geneva / Lyon / KEF

去程从日内瓦出发，坐 FlixBus 或火车 到 Lyon，再乘 easyJet LYS -> KEF，抵达雷克雅未克约 19:00。回程按 7月28日 18:00 KEF -> 23:50 LYS 准备，Lyon 当晚过夜。

行李额大家可以拼一下，2-3 人合买 1 件托运行李，把十天旅途的厚衣物、防水装备、公共物资和食物调料集中放托运。准备更周全，特别是我们大部分时候是在民宿做饭

路线

逆时针环岛 + 西峡湾

主线为 Reykjavík -> Hella -> Vík -> Höfn -> Egilsstaðir -> Mývatn/Húsavík/Akureyri -> Westfjords -> Snæfellsnes -> Reykjavík。目前想法是7月24日给观鲸单独留一天，7月25日坐长途车。

西峡湾两天路程较长。需要具体商议关于住宿和渡轮时间。

渡轮

Brjánslækur -> Stykkishólmur

7月26日按 Baldur ferry 的 Brjánslækur -> Stykkishólmur 带车渡轮准备。常见班次模式里有 12:00 / 18:00 方向船，实际出票前以 ferja.is 为准。

带车上船至少提前45-60分钟到港。不坐渡轮时，改走陆路（比坐船多1h），具体时间当天用 Google Maps 再确认。坐船大概人均60瑞左右
